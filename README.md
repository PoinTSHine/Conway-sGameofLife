# `Conway's Game of Life`

一个基于`HTML`、`CSS`和`JavaScript`实现的康威生命游戏（`Conway's Game of Life`）。

## 项目简介

康威生命游戏是由英国数学家约翰·康威（`John Conway`）在1970年设计的一种元胞自动机（`cellular automaton`）。它展示了如何从简单的规则中涌现出复杂的模式和行为。

这个项目实现了一个交互式的康威生命游戏，具有现代化的界面设计和流畅的用户体验。

## 游戏规则

游戏在一个二维网格上进行，每个格子（细胞）有两种状态：存活（`alive`）或死亡（`dead`）。每个细胞的状态会根据其周围8个邻居的状态，按照以下规则在每一代发生变化：

1. **生存规则**：如果一个活细胞周围有2个或3个活邻居，它将继续存活到下一代。
2. **死亡规则**：如果一个活细胞周围的活邻居数量少于2个（孤独）或多于3个（拥挤），它将在下一代死亡。
3. **繁殖规则**：如果一个死细胞周围恰好有3个活邻居，它将在下一代复活。

## 功能特点

- 🎮 **交互式网格**：点击单元格设置初始状态
- ▶️ **控制按钮**：开始、暂停、重置模拟
- 🎨 **现代化UI**：渐变背景、悬停效果和平滑动画
- 📱 **响应式设计**：自适应不同屏幕尺寸
- ⚙️ **可配置网格大小**：通过网页界面调整网格的行数和列数
- 📊 **状态反馈**：实时显示模拟状态

## 如何使用

1. **调整网格大小**：
   - 在"Rows"输入框中设置网格的行数（5-50之间）
   - 在"Cols"输入框中设置网格的列数（5-50之间）
   - 点击「`Apply Size`」按钮应用新的网格大小

2. **设置初始状态**：
   - 点击网格中的任意单元格，使其变为存活状态（蓝色）
   - 再次点击已存活的单元格，可使其变为死亡状态

3. **控制模拟**：
   - 点击「`Start`」按钮开始演化
   - 点击「`Pause`」按钮暂停演化
   - 点击「`Reset`」按钮重置网格

4. **观察演化**：
   - 观察细胞群体如何根据规则进行演化
   - 尝试创建不同的初始模式，观察它们的行为

## 技术实现

### 核心技术

- **HTML5**：页面结构和DOM操作
- **CSS3**：现代化样式、动画和响应式设计
- **JavaScript (ES6+)**：游戏逻辑和交互控制

### 主要功能模块

1. **网格初始化** (`initGrid`)
   - 动态创建指定大小的网格
   - 绑定单元格点击事件

2. **细胞状态管理** (`toggleCell`, `updateGridDom`)
   - 处理用户点击切换细胞状态
   - 更新DOM显示细胞状态

3. **演化逻辑** (`countLiveNeighbors`, `evolve`)
   - 计算每个细胞周围的活邻居数量
   - 应用康威生命游戏规则更新网格状态

4. **用户界面控制** (`bindEvents`)
   - 处理开始、暂停、重置按钮事件
   - 更新状态显示

5. **网格大小管理** (`getGridSize`, `applyGridSize`)
   - 从网页界面获取网格大小设置
   - 应用新的网格大小并重新初始化网格

### 视觉设计

- 渐变背景动画，营造科技感
- 半透明玻璃态按钮设计
- 活细胞的脉动效果
- 响应式网格布局

## 配置选项

### 网格大小

通过网页界面的输入框可以直接配置网格大小：

- **Rows**：设置网格的行数，范围为5-50
- **Cols**：设置网格的列数，范围为5-50

### 演化速度

演化间隔时间固定为200毫秒，这是平衡视觉效果和性能的理想设置。

## 关于康威生命游戏

康威生命游戏是一个零玩家游戏，这意味着它的演化完全由初始状态决定，不需要进一步的用户输入。它是复杂性科学中的经典示例，展示了简单规则如何产生复杂的涌现行为。

一些著名的模式包括：

- 滑翔机（`Glider`）：能够在网格上移动的模式
- 闪烁器（`Blinker`）：最简单的振荡器
- 脉冲星（`Pulsar`）：周期为3的振荡器
- 蜂巢（`Honeycomb`）：稳定的静态模式
